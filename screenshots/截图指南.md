## 需要截的 4 张图

| 文件名 | 截图内容 | 怎么截 |
|--------|---------|--------|
| `01-main.png` | 主界面全貌 | 打开 http://localhost:8501，截整个浏览器窗口，展示左侧配置面板 + 右侧输入区 + 示例模板 |

## 截图方法

### Windows 自带截图（推荐）
```
Win + Shift + S  →  框选区域  →  自动复制到剪贴板
```
然后粘贴到画图工具保存为 PNG，放到 `screenshots/` 文件夹。

### 或者直接用浏览器截图插件
- Chrome: 右键 → 检查 → Ctrl+Shift+P → 输入 "screenshot" → 选 "Capture full size screenshot"
- 或者安装插件: GoFullPage

## 放到 GitHub 上

把截图放进文件夹后，直接 `git add`、`git commit`、`git push` 就行。GitHub 会自动渲染 README 里的图片。

```bash
git add screenshots/
git commit -m "docs: 添加功能截图"
git push
```
